개요
======
프로젝트 이름: ATM
======
프로젝트 기간: 25.03.20 ~ 25.03.21
======

주요기능
- 로그인 / 회원가입 
- 입금 / 출금 / 송금
- 데이터 저장 및 로드 

게임 설명
처음 게임을 시작하면 로그인 창이 뜬다. 로그인 창에는 아이디와 비밀번호를 입력할 수 있는 필드와 로그인 버튼, 회원가입 버튼이 있다. 아이디와 비밀번호 필드에 하나라도 입력하지 않거나 잘못 입력했을 경우 정보를 다시 확인하라는 입력창이 뜨게 되고 모두 다 정확하게 입력하면 로그인 할 수 있다.
